# jigsaw-game-robot

## Hardware
robot: Franka panda

camera: Realsense 400 series

## Enviroment set up
pre prepare
1.pc with ubuntu
2.ros
3.Franka ros package
4.Realsense SDK, ros package and librealsense2

clone this repository to your work space
```sh
$ cd ~/catkin_ws/src
$ git clone https://github.com/gesheng199832/jigsaw-game-robot.git
$ catkin build
```
